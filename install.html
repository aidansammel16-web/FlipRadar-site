Install and Use FlipRadar

This page shows you how to install FlipRadar, set up your first search, and optionally connect Telegram alerts.

Install from Chrome Web Store View Pro Features
Replace CHROME_STORE_URL_HERE with your real Chrome Web Store listing URL once the extension is published.

1. Install FlipRadar

  1. Open the FlipRadar listing in the Chrome Web Store.
  2. Click Add to Chrome.
  3. Confirm the installation when Chrome asks.
  4. Pin FlipRadar to your browser toolbar so it is easy to access.
After installing, click the FlipRadar icon in your browser toolbar to open the popup.

2. Add Your First Marketplace Search

  1. Go to Facebook Marketplace.
  2. Search for the kind of product you want to monitor.
  3. Apply any Marketplace filters you want.
  4. Copy the full URL from your browser address bar.
  5. Open FlipRadar.
  6. Paste the Marketplace URL into the search field.
  7. Optionally set a max price.
  8. If you are on Pro, you can also enter keywords.
  9. Click Save Search.
Free users can monitor 1 search with 5 minute checks. Pro users get unlimited searches, keywords, and faster alerts.

3. Configure Your Settings

Minimum score

Check interval

This determines how often FlipRadar refreshes your saved searches.

  • Free: 5 minute minimum

4. Set Up Telegram Alerts (Optional)

FlipRadar can send alerts through your own Telegram bot. You do not need Telegram to use FlipRadar, but it is useful if you want mobile notifications.

Step A — create a Telegram bot

  1. Open Telegram.
  2. Search for BotFather.
  3. Start a chat with BotFather.
  4. Send this command:
/newbot
  1. Follow the prompts to choose a bot name and username.
  2. BotFather will give you a bot token.
  3. Copy and save that token somewhere safe.

Step B — get your chat ID

  1. Send a message to your new bot in Telegram.
  2. Open this URL in your browser, replacing YOUR_BOT_TOKEN with your real token:
https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
  1. Find the number listed under chat → id.
  2. Copy that number. That is your Telegram chat ID.

Step C — add both values to FlipRadar

  1. Open FlipRadar.
  2. Open full settings.
  3. Paste in your Telegram bot token.
  4. Paste in your Telegram chat ID.
  5. Save your settings.
Keep your Telegram bot token private. Anyone with that token can interact with your bot.

5. How to Use FlipRadar Effectively

  • Use very specific Marketplace searches for the best results.
  • Set a max price so you only see real opportunities.
  • Use keywords on Pro to narrow down listings even further.
  • Keep FlipRadar pinned so you can quickly update searches.
  • Check alerts quickly because good Marketplace deals disappear fast.
